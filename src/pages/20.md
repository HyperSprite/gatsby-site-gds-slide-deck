## Create the `gds-config.json` file


* Create a `gds-config.json` file in the root of your project directory


* Copy and paste this example into your file


```json
{
  "outputRoot": "./src/",
  "folders": [
    {
      "localFolder": "pages/",
      "folderId": "YOUR_FOLDER_KEY"
    },
    {
      "localFolder": "pages/images/",
      "folderId": "YOUR_FOLDER_KEY",
      "docType": "media"
    }
  ]
}
```
